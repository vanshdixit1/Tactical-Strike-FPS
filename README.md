# Tactical Strike: Counter Ops 🔫

**Tactical Strike** is a high-octane, browser-based tactical first-person shooter built with Three.js. Experience intense round-based combat, strategic weapon purchasing, and challenging AI bots in a modern, neon-lit environment.

## 🚀 Play Now
The game is deployed and ready to play at:
**[https://tactical-strike-fps.vercel.app](https://tactical-strike-fps.vercel.app)**

---

## 🎮 Gameplay Features

### ⚔️ Combat System
- **Advanced Ballistics**: Realistic recoil patterns, bullet spread, and damage falloff.
- **Weapon Variety**: 
  - **Pistols**: USP-S, Desert Eagle
  - **Rifles**: AK-47, M4A1
  - **SMGs**: MP5
  - **Snipers**: AWP (with functional scope system)
- **Economy System**: Earn money through kills and round wins to buy better gear in the **Buy Menu [B]**.

### 🤖 Intelligent AI Bots
- Bots navigate the map using a custom waypoint system.
- Adaptive difficulty levels (Recruit, Veteran, Elite).
- Bots can use cover, aim, and engage the player strategically.

### 🗺️ Tactical HUD & UI
- **Real-time Minimap**: Track enemy positions and map layout with a high-tech radar scanner.
- **Kill Feed**: Stay updated on eliminations and headshots.
- **Dynamic HUD**: Monitor health, armor, ammo, and round progression.
- **Scoreboard**: Track your performance throughout the mission.

### 🔊 Immersive Audio
- Procedural footstep sounds based on movement speed.
- Unique weapon sounds for firing, reloading, and impacts.
- Spatial audio for a more immersive tactical experience.

## ⌨️ Controls

| Action | Key |
| :--- | :--- |
| **Movement** | `W` `A` `S` `D` |
| **Look Around** | `Mouse` |
| **Shoot** | `Left Click` |
| **Scope / Zoom** | `Right Click` |
| **Reload** | `R` |
| **Switch Weapon** | `1` `2` `3` |
| **Sprint** | `Shift` |
| **Jump** | `Space` |

## 🛠️ Technical Details

- **Engine**: [Three.js](https://threejs.org/) (WebGL)
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics.
- **Fonts**: Rajdhani & Orbitron (Google Fonts).
- **Architecture**: Modular JavaScript with dedicated systems for Weapons, AI, Mapping, and Audio.

## 🛠️ Installation & Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vanshdixit1/Tactical-Strike-FPS.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser.
3. **Local Server (Optional)**:
   For the best experience (and to avoid CORS issues with some assets), run a local server:
   ```bash
   npx serve .
   ```
